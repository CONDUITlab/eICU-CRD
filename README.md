# eICU-CRD
This is the R code for descriptive analysis of the eICU Collaborative Research Database

The SQL code embedded in the R code (as comments) can be used to download the data tables used in their entirety, using Google BigQuery. Tables are then uploaded directly into R for further analysis.

June 1, 2020
This code is provided as supplementary material for a manuscript describing the eICU-CRD submitted to Critical Care Medicine.
